GOF2 - Használj objektum-összetételt öröklés helyett, ha csak lehet!

Feladat:
GOF1\Matrix\Library\MAF.EKE.GOF1 projekt segítségével készítsünk egy olyan szolgáltatást(dll), ami a mátrixról visszaad minden olyan információt egy string-ben, amit az "õs" szolgáltatásai tudnak, valamint magát a mátrixot is kiírja. 
A szolgáltatás sebességre optimalizált legyen.
Tartsuk be a GOF2 elvet valamint az elõzõ példában megtanult GOF1 elvet is!
Ha ezzel kész vagyunk, készítsünk egy konzolos alkalmazást, amivel ezeket az információkat kiírjuk a konzolra.

Ha nem boldogulunk a feladattal, akkor a következõ példákat nézzük át:
 - A Library könyvtár MAF.EKE.GOF2 projektje a szolgáltatást valósítja meg.
 - A Program könyvtár MAF.EKE.GOF2.MatrixExample a konzolos kiírást mutatja be.

A GOF1 és GOF2 konzolos példaprogramot hasonlítsuk össze, hogy mennyivel lett egyszerûbb a mátrix információinak kiíratása.

Mindenkinek jó gyakorlást!

Megj: A MAF.EKE.GOF1 projektet az eredeti helyérõl húzzuk be. Ha a fejlesztésekhez jól megtervezett könyvtárszerkezeteket használunk, akkor minden gond nélkül behúzhatunk más Solution projektjeit.
